File: MakeWaves_ReadMe.txt
Purpose: Describe MakeWaves.py script for creation of ADALM2000 Scopy Oscilloscope CSV export files
Author: Ron at BiophysicsLab.com
Initial Date: 10/14/2019

To use program:

1) Update your target wave generator such as a sine wave or square wave within the code
2) Execute program

Program output:

A csv file suitable for use by BiophysicsLab.com's M2KScopePlot.py (a.k.a. the main program)

Final steps:

1) Launch M2KScopePlot.py
2) Using the menu, point the program to the location of the csv file
3) Follow the remaining menu items to create the desired plot, png, and txt files
